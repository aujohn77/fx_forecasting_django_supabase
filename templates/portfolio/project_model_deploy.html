FX Model Deployment Demo

This project is a full FX forecasting pipeline built with Django, Supabase (Postgres), and Render.

  • Ingests daily FX rates from the Frankfurter API.
  • Stores 10+ years of history in Supabase.
  • Runs multiple forecasting models (Python and R) via scheduled jobs.
  • Serves interactive dashboards for Market and Forecasts.
